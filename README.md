**Data Engineer Technical Assessment**

This technical assessment is designed as a take-home assignment to evaluate a candidate's skills against the requirements for the Consultant Data Engineer role, as outlined in the Data Engineer JD.

**Scenario**

You are a Data Engineer in a product team that is building a conversational chatbot service that will be incorprated into many existing websites in Singapore.

The conversation logs between the agent and the user are to be preserved for further evaluations of the agents' response to user queries and identify potential product specific questions or identify gaps.

**You will be required:**

Create specifications to store the conversation between the agent and users.
Design small data pipeline to ingests raw event data, transforms it into analytics-ready tables, and produces metrics for stakeholders.
Provide inputs to business users in quantifying product success via SQL analysis and documentation.
You may use any language and tooling you are comfortable with (e.g. Python + SQL, dbt, Spark, DuckDB, etc.), as long as the outputs and instructions are clear and reproducible.
